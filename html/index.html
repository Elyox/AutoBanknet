Bienvenue dans la
BankAutobahn

Ce projet a pour but de mettre à disposition du grand public un service de banque en ligne. Nous nous somme penché sur la création d’un site web, d’un serveur et d’une base de données.
Nous avons choisi de créer se projet car il implique la communication à la fois entre deux ordinateurs à travers le web mais également entre différents langages des programmation.

Nous nous sommes répartis les taches suivantes :
- Création d’une interface web → Lenz
- Mise en place d’une base de données → Florian
- Envoi de requêtes d’un client à serveur et traitement des réponses → Lenz
- Traitement des requêtes client → Fidel
- Interrogation de la base de donnée → Florian
- Création d’une application mobile → Florian
- Sécuriser les communications → Lenz
- « Hashage » des mots de passe → Fidel

Nous avons utilisé les langages, outils et bibliothèques suivants :
- Le javascript pour le client et le serveur et permet la communication entre les deux.
- Node.js qui est un outil permettant la mise en place d’un serveur HTTP en javascript. On utilise les bibliothèques mysql et bcrypt.
- Le HTML/CSS pour créer une interface web. On utilise en plus l’outil de développement Bootstrap de Twitter qui permet une mise en page responsive (qui s’adapte à tous les écrans) plus aisée.
- Le SQL pour communiquer avec la base de donnée.
- Le Java pour l’application mobile. On utilise la bibliothèque WebView.
